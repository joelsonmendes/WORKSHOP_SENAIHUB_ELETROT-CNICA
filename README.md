# Aulão Prático — Fundamentos de Sistemas Elétricos

Projeto PWA em HTML, CSS e JavaScript para acompanhar o workshop prático de Fundamentos de Sistemas Elétricos.

## Módulos

1. Dia 1 — Medições Elétricas
2. Dia 2 — Corrente Alternada
3. Dia 3 — Impedância RL/RC/RLC
4. Dia 4 — Potência em CA
5. Dia 5 — Correção de Fator de Potência
6. Dia 6 — Ensaio com Megômetro
7. Dia 7 — Eletrônica Aplicada
8. Dia 8 — Desafio Integrador

## Logo

Coloque a logo em:

assets/logo-senai-hub.webp

## Como usar

Abra o arquivo:

index.html

Ou publique em GitHub Pages, Vercel ou servidor local.

## Observação

Os dados são salvos no navegador usando localStorage.
